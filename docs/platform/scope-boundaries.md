# Platform Scope Boundaries

## In Scope

- selected Standards III-VI topics;
- Science and Mathematics first;
- Geography optional/secondary;
- topic screening;
- automated curriculum review as assistive only;
- human topic selection;
- topic brief generation;
- generation provenance;
- artifact inspection;
- expert review;
- rubric feedback;
- export;
- replay/provenance explanation;
- WhatsApp invitation/reminder channel.

## Out of Scope

- full national curriculum coverage;
- classroom deployment;
- learner-outcome measurement;
- student accounts;
- large-scale LMS features;
- foundation model training;
- mandatory fine-tuning;
- fully automated curriculum judgment;
- LLM auto-selection of topics;
- WhatsApp-only evaluation;
- production deployment.

## Boundary Rule

Every feature must map to a research objective, evaluation activity, evidence artifact, or reproducibility requirement. Features that do not satisfy that rule should be deferred.
