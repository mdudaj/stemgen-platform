# Product

## Purpose

Kisomo supports the creation, inspection, comparison, and evaluation of
curriculum-aligned STEM animation and narration prototypes for Tanzanian primary
education.

The platform is a research instrument. It must preserve evidence about how
artifacts were produced and how experts judged their feasibility.

## Primary Users

- research operators configuring and running workflows
- evaluators reviewing artifacts and submitting structured feasibility feedback

## Core Outcomes

1. An operator can select curriculum material and launch a reproducible run.
2. The system records provenance sufficient to explain the run.
3. Operators can inspect artifacts, run status, timing, and outputs.
4. Evaluators can submit structured feasibility review data.
5. The system can export artifacts, provenance, and review records for analysis.

## Current Rebuild Boundary

The current directory is at baseline-harness stage: auth, shell, assets, and
verification scaffolding. Domain workflows are pending review before restore.

## Milestone 22.5 Evaluation Readiness

The platform is both a media-generation platform and a research instrument. Evaluation-readiness work must connect every feature to a research objective, evaluation activity, evidence artifact, or reproducibility requirement.

Milestone 22.5 keeps implementation bounded to documentation, ADRs, schemas, requirements, and vertical slices. It does not restore the full curriculum or generation workflow yet.
